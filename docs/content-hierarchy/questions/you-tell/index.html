      🪞 自查 #

      数据结构 #

      示例如下：

      kind: YOU_TELL
config:
	stems:
		- kind: TEXT
		  text: >
		    我们称一些能瞬间理解、记忆的内容为瞬间可检索，人脑瞬间可检索的容量十分有限。那么如果你想让你脑中可
		    检索的信息容量变大，可以怎么做？
	explains:
		- kind: TEXT
		  text: >
		  	尽管脑中瞬间可检索的容量有限，但我们可以以这些瞬间可检索的内容为线索，建立它们与其它信息的联系，
		  	利用间接检索的方式扩大脑中的可检索信息容量。

      自查类型的问题没有额外字段。
